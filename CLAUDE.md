## Common Commands

```bash
# Apply configuration changes
sudo nixos-rebuild switch --flake .#nixos

# Update flake inputs
nix flake update
```

## Architecture

NixOS configuration using Nix Flakes + Home Manager for a single-user desktop (i3, aarch64-linux).

- **`flake.nix`** — Entry point. Inputs: nixpkgs stable (25.05), nixpkgs-unstable, home-manager.
- **`configuration.nix`** — System-level: services, packages, desktop, audio (PipeWire), networking, locale.
- **`home.nix`** — User-level via Home Manager: packages, dotfiles, shell (zsh/oh-my-zsh), GTK theme.
- **`hardware-configuration.nix`** — Auto-generated by NixOS. Do not edit manually.
- **`dotfiles/`** — Raw config files linked via Home Manager (`home.file` / `xdg.configFile` in `home.nix`).

Packages needing newer versions come from `nixpkgs-unstable` (referenced as `pkgs.unstable.<name>`).
